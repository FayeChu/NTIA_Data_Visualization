All the Census datasets involved in this Capstone are provided by the US Census Bureau and it provides a website for people to retrieve various Census data files from - the American FactFinder.

Demographics measurements from Census include:
  • 2016 ACS* 5-year Estimates Dataset: Educational Attainment (S1501), Median Income in the Past 12 Months in 2016 Inflation-Adjusted Dollars (S1903), Total Population (B01003), Poverty Rate (S1701)
  • Decennial Census program's 2010 Census Summary File Dataset: Population, Housing Units, Area, and Density
*: American Community Survey program's 2016 ACS 5-year estimates dataset
  • 2010 Census Geographic Entity Tallies by State and Type - Number of counties in each state
  • Census blocks by State
